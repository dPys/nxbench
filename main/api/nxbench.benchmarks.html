nxbench.benchmarks.benchmark.process_algorithm_params(params)[source]¶

Process and separate algorithm parameters into positional and keyword arguments.

Parameters:

params (dict[str, Any]) – A dictionary of algorithm parameters, where keys can indicate either positional or keyword arguments.

Returns:

A tuple containing a list of positional arguments and a dictionary of keyword arguments.

Return type:

tuple[list[Any], dict[str, Any]]

Notes

Parameters prefixed with an underscore (“_”) are treated as positional arguments. If a parameter value is a dictionary containing a “func” key, the function is imported dynamically.
